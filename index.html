Aqui você vai codificar ou decodificar sua mensagem usando dois métodos.

O método Base64: possui um número de 64 caracteres servindo como base de um texto.

A variação entre os caracteres vão de “A-Z”, “a-z”, “0-9”, “+”, “/” e “=” .

resultado: TmFkYSBNYWlzIEltcG9ydGE = Nada Mais Importa;


Método Cifra de Cesar: é um tipo de cifra de substituição na qual cada letra no texto simples
é deslocada para um certo número de lugares abaixo do alfabeto.
Resultado: Jvumph uv wyvjlzzv = Confia no processo;
